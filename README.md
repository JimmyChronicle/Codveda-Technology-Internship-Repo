# Data Analytics Internship Projects | Codveda Technology

This repository showcases all the hands-on data analytics projects I completed during my **Data Analytics Internship at Codveda Technology**.

I'm documenting my journey as I build practical skills — from cleaning raw data to performing advanced analysis and creating insightful visualizations.

## 📋 What’s Inside

### Level 1 (Basic)
- **Data Cleaning & Preprocessing**: Handled missing values, removed duplicates, and standardized formats using pandas.
- **Exploratory Data Analysis (EDA)**: Calculated summary statistics, found correlations, and explored data patterns.
- **Basic Data Visualization**: Created bar plots, line charts, and scatter plots with Matplotlib & Seaborn.

### Level 2 (Intermediate)
- **Regression Analysis**: Built and evaluated linear regression models using scikit-learn.
- **Time Series Analysis**: Detected trends and seasonality, performed decomposition, and applied moving average smoothing.
- **K-Means Clustering**: Implemented clustering with feature scaling and visualized results using the elbow method.

### Level 3 (Advanced)
- **Predictive Modeling (Classification)**: Built models (Logistic Regression, Decision Trees, Random Forest) with hyperparameter tuning.
- **Interactive Dashboards**: Created dynamic dashboards in Power BI / Tableau.
- **NLP - Sentiment Analysis**: Analyzed text data using NLTK/TextBlob and generated word clouds.

## 🛠️ Tech Stack
- **Python**
- **pandas**, **NumPy**
- **Matplotlib**, **Seaborn**
- **scikit-learn**, **statsmodels**
- **NLTK**, **TextBlob**
- **Power BI**
